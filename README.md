# charging-stand-clock
Code for a custom WaveShare eInk display clock next to my charging stand

# Compiling with arduino-cli
If you're like me and don't want to compile using the Arduino IDE, here's the command I used for reference

Compile with: `arduino-cli compile -b esp32:esp32:esp32`

Upload with: `arduino-cli upload -p COM3 -b esp32:esp32:esp32`
